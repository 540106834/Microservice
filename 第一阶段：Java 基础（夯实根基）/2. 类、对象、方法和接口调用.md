 # **Java 中的类、对象、方法和接口调用** 

## 一、类（Class）

1. **定义**：类是对象的模板或者蓝图，描述一类事物的属性（成员变量）和行为（方法）。
2. **作用**：封装数据和行为，实现代码复用。
3. **基本结构**：

```java
public class Person {
    // 属性（成员变量）
    String name;
    int age;

    // 方法（行为）
    void sayHello() {
        System.out.println("Hello, my name is " + name);
    }
}
```

* **类成员**：

  * **变量**：描述对象的状态（属性）
  * **方法**：描述对象的行为（功能）

---

## 二、对象（Object）

1. **定义**：对象是类的实例，是类在内存中的具体存在。
2. **创建对象**：

```java
Person p1 = new Person(); // 创建对象
p1.name = "Alice";        // 设置属性
p1.age = 20;
p1.sayHello();            // 调用方法
```

3. **对象特点**：

* 每个对象独立拥有自己的属性值
* 对象方法可以操作对象属性

---

## 三、方法（Method）

1. **定义**：方法是类的行为，用来封装可复用的逻辑。

2. **方法类型**：
   | 类型 | 特点 | 示例 |
   |------|------|------|
   | 实例方法 | 属于对象，需要对象调用 | `p1.sayHello()` |
   | 静态方法 | 属于类，可以直接调用 | `Math.max(1,2)` |
   | 带返回值方法 | 有返回值，需要 `return` | `int sum() {...}` |
   | 无返回值方法 | 使用 `void` | `void print()` |

3. **示例**：

```java
public class Calculator {
    // 无返回值
    void printHello() {
        System.out.println("Hello!");
    }

    // 有返回值
    int add(int a, int b) {
        return a + b;
    }

    // 静态方法
    static int multiply(int a, int b) {
        return a * b;
    }
}

public class Test {
    public static void main(String[] args) {
        Calculator calc = new Calculator();
        calc.printHello();
        System.out.println(calc.add(3, 4));
        System.out.println(Calculator.multiply(2, 5));
    }
}
```

## 四、接口（Interface）

1. **定义**：接口是抽象的类，是功能的规范，用来描述对象应该做什么，但不实现具体逻辑。
   
>   接口只规定“你必须会什么”，实现类可以自由“长肉”。
```
运维 / API 场景总结 
接口:定义能力、规范、约定（健康检查、日志、配置读取）

实现类:保存状态、封装逻辑、写工具方法、接中间件

接口像一张“能力清单”，
实现类像一个背着工具箱的工程师。
```
2. **特点**：

* 只定义方法签名（Java 8 之后可以有默认实现）
* 类可以实现接口（`implements`），实现接口方法

3. **语法示例**：

```java
// 定义接口
interface Animal {
    void eat();
    void sleep();
}

// 实现接口
class Dog implements Animal {
    public void eat() {
        System.out.println("Dog is eating");
    }
    public void sleep() {
        System.out.println("Dog is sleeping");
    }
}

// 测试
public class TestInterface {
    public static void main(String[] args) {
        Animal a = new Dog(); // 接口引用指向实现类对象
        a.eat();
        a.sleep();
    }
}
```

**输出**：

```
Dog is eating
Dog is sleeping
```

---

## 五、类与对象、方法、接口的关系总结

1. **类** = 模板
2. **对象** = 类的实例
3. **方法** = 类/对象的行为
4. **接口** = 功能规范，定义必须实现的方法
5. **关系图**：

```
接口 -> 实现类 -> 创建对象 -> 调用方法
```

## 六、综合练习案例：健康检查服务模拟

```java
// 接口定义
interface HealthService {
    String getStatus();
    void check();
}

// 类实现接口
class MyHealthService implements HealthService {
    private String status;

    public MyHealthService() {
        this.status = "DOWN";
    }

    public String getStatus() {
        return status;
    }

    public void check() {
        // 模拟健康检查逻辑
        status = "UP";
        System.out.println("服务状态检查完成: " + status);
    }
}

// 测试类
public class TestHealth {
    public static void main(String[] args) {
        HealthService service = new MyHealthService(); // 接口引用
        System.out.println("初始状态: " + service.getStatus());
        service.check();
        System.out.println("最终状态: " + service.getStatus());
    }
}
```

**输出**：

```
初始状态: DOWN
服务状态检查完成: UP
最终状态: UP
```

---

这个练习综合了：

* 类和对象
* 方法调用（实例方法）
* 接口调用（通过接口引用调用实现类）
