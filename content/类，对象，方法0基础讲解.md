# 一、先一句话打底

> **类**：说明书 / 图纸
> **对象**：照着说明书造出来的“实物”
> **方法**：这个实物“能做的动作”

记住这三句话，后面所有内容都会往这三句话上靠。

---

# 二、什么是「类」（Class）

### 用人话说

**类不是东西，是“定义”**

就像：

* 菜谱不是菜
* 房屋设计图不是房子
* 游戏角色设定不是角色本人

---

### Java 中的类

```java
class Person {
    String name;
    int age;

    void sayHello() {
        System.out.println("你好，我叫 " + name);
    }
}
```

我们慢慢拆：

#### 1️ `class Person`

> 定义了一个“人”的模板，名字叫 Person

#### 2️ `String name;`

> 人有一个“名字”

#### 3️ `int age;`

> 人有一个“年龄”

 这些叫 **属性**（变量）

#### 4️ `void sayHello() { ... }`

> 人可以“打招呼”

 这叫 **方法**

---

### 此时重要的一句话

> **现在世界上还没有“人”**

只有一个“人类说明书”。

---

# 三、什么是「对象」（Object）

### 用人话说

**对象 = 类造出来的真实东西**

---

### 用类造对象

```java
Person p = new Person();
```

逐字翻译：

| 代码         | 人话             |
| ---------- | -------------- |
| `Person`   | 我用的是“人”这个类型    |
| `p`        | 我给这个人起个名字叫 p   |
| `new`      | 新造一个           |
| `Person()` | 按照 Person 的说明书 |

 **这一步，世界上第一次真的有“人”了**

---

### 给对象赋值

```java
p.name = "张三";
p.age = 18;
```

> 给这个“具体的人”设置名字和年龄

---

# 四、什么是「方法」（Method）

### 用人话说

**方法 = 对象能做的事情**

比如人可以：

* 说话
* 走路
* 吃饭

---

### 调用方法

```java
p.sayHello();
```

发生了什么？

> 让 **p 这个人** 执行 “打招呼” 这个动作

屏幕输出：

```
你好，我叫 张三
```

---

# 五、连起来看一遍（完整示例）

```java
class Person {
    String name;
    int age;

    void sayHello() {
        System.out.println("你好，我叫 " + name);
    }
}

public class Demo {
    public static void main(String[] args) {

        Person p = new Person();   // 造对象
        p.name = "张三";           // 设置属性
        p.age = 18;

        p.sayHello();              // 调用方法
    }
}
```

# 六、非常关键的 3 个「新手必懂点」

## ① 类 ≠ 对象（这是 80% 新手的第一坑）

```java
Person p1 = new Person();
Person p2 = new Person();
```

👉 这是 **两个人**
改 `p1.name` 不会影响 `p2.name`

---

## ② 方法必须属于某个类

❌ 错误思维：

> 我想写个方法

✅ 正确思维：

> 我想让**某个东西**能干什么

所以 Java 里：

> **方法不能漂浮在空气中**

---

## ③ 对象不创建，方法不能用

```java
Person.sayHello(); // ❌ 错
```

```java
Person p = new Person();
p.sayHello();      // ✅ 对
```

> **方法描述能干嘛**
