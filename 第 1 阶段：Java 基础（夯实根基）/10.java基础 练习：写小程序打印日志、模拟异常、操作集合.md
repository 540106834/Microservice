太好了，我们可以做一个 **综合练习**，把 Java 基础知识整合在一个小程序里：**打印日志、模拟异常、操作集合**。我帮你设计一个小示例，涵盖 **日志框架、异常处理、集合操作**。

---

## 练习目标

1. 使用 **SLF4J + Logback** 打印日志
2. 模拟异常并捕获，输出堆栈
3. 使用 **List、Set、Map** 操作数据
4. 使用 **泛型** 保证类型安全

---

## Maven 依赖（pom.xml）

```xml
<dependencies>
    <!-- 日志门面 -->
    <dependency>
        <groupId>org.slf4j</groupId>
        <artifactId>slf4j-api</artifactId>
        <version>2.0.9</version>
    </dependency>
    <!-- Logback 实现 -->
    <dependency>
        <groupId>ch.qos.logback</groupId>
        <artifactId>logback-classic</artifactId>
        <version>1.4.11</version>
    </dependency>
</dependencies>
```

---

## Logback 配置（logback.xml）

```xml
<configuration>
    <appender name="STDOUT" class="ch.qos.logback.core.ConsoleAppender">
        <encoder>
            <pattern>%d{HH:mm:ss} [%thread] %-5level %logger{36} - %msg%n</pattern>
        </encoder>
    </appender>

    <root level="INFO">
        <appender-ref ref="STDOUT"/>
    </root>
</configuration>
```

## Java 综合示例

```java
import org.slf4j.Logger;
import org.slf4j.LoggerFactory;

import java.util.*;

public class JavaPractice {

    private static final Logger logger = LoggerFactory.getLogger(JavaPractice.class);

    public static void main(String[] args) {
        logger.info("程序开始");

        // 1. List 操作
        List<String> fruits = new ArrayList<>();
        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Orange");
        fruits.add("Apple"); // List 可重复
        logger.info("水果列表: {}", fruits);

        // 2. Set 操作
        Set<String> uniqueFruits = new HashSet<>(fruits); // 自动去重
        logger.info("去重后的水果集合: {}", uniqueFruits);

        // 3. Map 操作
        Map<String, Integer> fruitCount = new HashMap<>();
        for (String fruit : fruits) {
            fruitCount.put(fruit, fruitCount.getOrDefault(fruit, 0) + 1);
        }
        logger.info("水果出现次数统计: {}", fruitCount);

        // 4. 模拟异常并捕获
        try {
            int result = divide(10, 0); // 会抛异常
            logger.info("计算结果: {}", result);
        } catch (ArithmeticException e) {
            logger.error("捕获异常: 除数不能为零", e);
        }

        logger.info("程序结束");
    }

    // 方法：除法
    public static int divide(int a, int b) {
        return a / b; // b=0 会抛出 ArithmeticException
    }
}
```

---

### 程序功能说明

1. **日志输出**：`INFO` 级别输出流程，`ERROR` 级别输出异常
2. **集合操作**：

   * `List` 可以重复
   * `Set` 自动去重
   * `Map` 统计出现次数
3. **异常处理**：

   * `try/catch` 捕获除零异常
   * `logger.error` 打印堆栈信息
4. **泛型**：

   * `List<String>`、`Set<String>`、`Map<String, Integer>` 保证类型安全
