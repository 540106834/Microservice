# **Java 的基础类与对象，以及封装、继承、多态**


## 1️ 类与对象

**类（Class）**：抽象模板，描述一类对象的属性（成员变量）和行为（方法）。
**对象（Object）**：类的实例，有具体的属性值和行为。

**示例：**

```java
// 类
public class Person {
    // 成员变量（属性）
    String name;
    int age;

    // 构造器
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // 方法（行为）
    public void sayHello() {
        System.out.println("Hello, my name is " + name);
    }
}

// 使用类创建对象
public class Main {
    public static void main(String[] args) {
        Person p = new Person("Alice", 25); // 创建对象
        p.sayHello();                       // 调用方法
    }
}
```

## 2️ 封装（Encapsulation）

**封装**就是把对象的属性私有化，并通过 getter/setter 方法提供访问权限，从而保护对象数据。

**特点：**

* 属性通常用 `private` 修饰
* 对外提供 `public` 方法访问属性
* 可以控制读写权限，增加数据安全性

**示例：**

```java
public class Person {
    private String name; // 私有属性
    private int age;

    public Person(String name, int age) {
        this.name = name;
        setAge(age); // 可以在 setter 内验证
    }

    // getter
    public String getName() {
        return name;
    }

    // setter
    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        if (age >= 0) { // 简单验证
            this.age = age;
        }
    }
}
```

## 3️ 继承（Inheritance）

**继承**是指一个类（子类）可以继承另一个类（父类）的属性和方法，从而实现代码复用和层次结构。

**特点：**

* 使用 `extends` 关键字
* 子类可以重用父类方法和属性
* 可以 **覆盖（override）** 父类方法

**示例：**

```java
// 父类
public class Animal {
    public void eat() {
        System.out.println("Animal is eating");
    }
}

// 子类
public class Dog extends Animal {
    // 重写 eat 方法
    @Override
    public void eat() {
        System.out.println("Dog is eating bones");
    }

    public void bark() {
        System.out.println("Woof!");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog dog = new Dog();
        dog.eat();  // 输出: Dog is eating bones
        dog.bark(); // 输出: Woof!
    }
}
```

## 4️ 多态（Polymorphism）

**多态**是指同一个操作作用于不同对象，可以表现出不同的行为。
在 Java 中，多态的实现依赖于 **继承 + 方法重写 + 父类引用指向子类对象**。

**特点：**

* 编译时类型是父类，运行时对象是子类
* 调用重写方法时，会执行子类实现

**示例：**

```java
Animal a1 = new Animal();
Animal a2 = new Dog(); // 父类引用指向子类对象

a1.eat(); // 输出: Animal is eating
a2.eat(); // 输出: Dog is eating bones —— 多态效果
```

> 注意：
>
> * 多态只能调用父类或接口中声明的方法
> * 子类特有的方法无法通过父类引用直接调用，需要向下转型

**向下转型示例：**

```java
Dog dog = (Dog) a2;
dog.bark(); // 输出: Woof!
```

## 5️ 总结表格

| 特性   | 定义                   | 关键点                     |
| ---- | -------------------- | ----------------------- |
| 类/对象 | 类是模板，对象是实例           | new 创建对象，调用方法           |
