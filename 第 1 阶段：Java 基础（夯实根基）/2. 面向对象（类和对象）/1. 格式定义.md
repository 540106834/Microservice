好，这一块是 **Java 的“语法骨架”**。把它想成一栋房子：
**类是房子，变量是房里的东西，方法是房子的功能按钮** 

我按 **定义格式 → 示例 → 关键点** 给你一套清晰版。

---

## 一 类（Class）的定义格式

###  基本格式

```java
访问修饰符 class 类名 {
    // 成员变量
    // 构造方法
    // 成员方法
}
```

###  示例

```java
public class User {
}
```

###  规则

* 类名：**大驼峰**（User、OrderService）
* 一个 `.java` 文件：

  * **最多一个 public 类**
  * 文件名 = public 类名

---

## 二️ 变量（Variable）的定义格式

变量本质：**一块有名字的内存**

### 1 成员变量（类里的）

```java
访问修饰符 数据类型 变量名;
```

```java
public class User {
    String name;
    int age;
}
```

 特点：

* 有默认值（`0 / null / false`）
* 作用范围：整个对象

---

### 2️ 局部变量（方法里的）

```java
数据类型 变量名 = 初始值;
```

```java
public void sayHello() {
    int count = 10;
}
```

 特点：

* **必须初始化**
* 只在方法里有效

---

### 3️ 静态变量（类变量）

```java
static 数据类型 变量名;
```

```java
class User {
    static String TYPE = "USER";
}
```

 特点：

* 属于 **类本身**
* 所有对象共享一份

---

## 三️ 方法（Method）的定义格式

###  标准格式

```java
访问修饰符 返回值类型 方法名(参数类型 参数名) {
    方法体;
    return 返回值;
}
```

---

###  示例 1：无返回值

```java
public void printInfo() {
    System.out.println("hello");
}
```

---

###  示例 2：有返回值

```java
public int add(int a, int b) {
    return a + b;
}
```

---

###  示例 3：无参数

```java
public String getName() {
    return name;
}
```

---

###  示例 4：有多个参数

```java
public void update(String name, int age) {
    this.name = name;
    this.age = age;
}
```

---

## 四️ 构造方法（Constructor）

构造方法是 **特殊的方法**，用来创建对象。

###  格式

```java
访问修饰符 类名(参数) {
}
```

###  示例

```java
public User(String name, int age) {
    this.name = name;
    this.age = age;
}
```

 特点：

* 名字 = 类名
* **没有返回值（连 void 都没有）**
* new 对象时自动调用

---

## 五️ 放在一起的完整例子（重点）

```java
public class User {

    // 成员变量
    private String name;
    private int age;

    // 静态变量
    public static String TYPE = "USER";

    // 构造方法
    public User(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // 成员方法
    public void sayHello() {
        System.out.println("hello, my name is " + name);
    }

    public int getAge() {
        return age;
    }
}
```
